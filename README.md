## Find Pairs Summing

Tested on Unix based machines, using python 3.9+.

## Prerequesites
 - Install python 3.9 or later
 - Install pip
 - Install virtualenv
 - ```git clone https://github.com/diegomillan/app.git```

## Build & Requirements
 From the root folder of this repository, execute:
 ```bash
 make requirements
 ```
## Test the app
 From the root folder of this repository, execute:
 ```bash
 make tests
 ```

## Usage
 From the root folder of this repository, execute:
 ```bash
 make run
 ```
